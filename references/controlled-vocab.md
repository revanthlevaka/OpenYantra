# OpenYantra Controlled Vocabulary v2.12

## Project Status (Karma-Sthiti)
`Active` / `Paused` / `Completed` / `Cancelled`

## Task Status (Kartavya-Sthiti)
`Pending` / `In Progress` / `Done` / `Blocked`

## Goal Status (Sankalpa-Sthiti)
`Active` / `In Progress` / `Achieved` / `Abandoned`

## Resolved? (Nishpanna)
`Yes` / `No`

## Priority (Pradhanta)
`Critical` / `High` / `Medium` / `Low`

## Confidence (Nishchaya)
`High` / `Medium` / `Low` / `Inferred`

## Source (Strot)
`User-stated` / `Agent-observed` / `Agent-inferred` / `System`

## Importance (Pradhanta-Ank)
1–10 integer (default 5)

## Strength (Bal)
`Strong` / `Mild` / `Uncertain`

## Sentiment (Bhavana)
`Positive` / `Neutral` / `Negative` / `Complex`
