The Write Flow

Karma-Lekha → Chitragupta → Chitrapat

1
Agent submits Karma-Lekha
Any agent calls request_write() — the only permitted write path.
oy.add_project("My Film", status="Active", importance=9) # or: oy.request_write(WriteRequest(...))
2
Sanchitta enqueue — crash safety
WriteRequest written to sanchitta.json before commit. Auto-replays on crash.
# sanchitta.json — persists to disk immediately queue.enqueue(req) # before anything else
3
Raksha security scan
Every write scanned for injection, jailbreak, impersonation. Confirmed threats → quarantine. Suspicious → warn + flag.
scan = raksha.scan_write(req.fields, req.agent, req.sheet) if scan.should_block: return "quarantined" elif scan.should_warn: log_security(req, scan)
4
Chitragupta validates
Checks controlled vocabulary, admission rules (no noise), conflict detection (Vivada).
validation = ledger._validate(req) # checks Status, Priority, Importance, Source vocab # filters: "user said thanks", importance < 2, etc
5
Mudra seal — SHA-256 signature
Write sealed with cryptographic signature before commit.
mudra = "sha256:" + hashlib.sha256(payload).hexdigest()[:32]
6
Commit to Chitrapat
Write to .ods file. VidyaKosha auto-syncs. Agrasandhanī records permanently.
ledger._commit(req, mudra) vidyakosha.sync(chitrapat_path) # index updated ledger._log_audit(req, "written", signature=mudra)

CLI Reference

yantra — the command line interface

yantra bootstrap
Interview-driven cold start — populates Chitrapat via conversation. No empty spreadsheets.
yantra ui [port]
Browser dashboard at http://localhost:7331. Today · Inbox · Loops · Timeline · Conflicts · Security.
yantra doctor
System health check — Python version, packages, port, permissions, ODS integrity.
yantra health
Memory stats — open loops, inbox pending, corrections, stale projects, file size.
yantra inbox "text"
Quick capture to Inbox without categorisation. Route later with yantra route.
yantra digest
Daily proactive summary — urgent loops, stale projects, memory insights, past memory.
yantra telegram
Start Telegram bot. Any message → Inbox. /loop /task /goal /health /digest commands.
yantra security
Full Chitrapat security audit — injection detection, Mudra verification, threat report.
yantra loops
List all unresolved Open Loops (Anishtha) with priority.
yantra diff
Belief contradiction check — detect evolved or conflicting beliefs across the Vishwas sheet.
yantra ttl
Check for expired Open Loops past their TTL_Days threshold.
yantra open
Open Chitrapat in LibreOffice for direct editing (Dharma-Adesh).
